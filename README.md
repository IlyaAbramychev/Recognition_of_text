# Распознавание текста на изображении с использованием нейросетевых методов

## **Описание**
Проект посвящён исследованию распознавания текста на изображениях с использованием классического метода Tesseract OCR и современных нейросетевых подходов (CRNN, Transformer). Основная цель — сравнить эффективность этих методов и исследовать влияние предобработки изображений на результаты распознавания текста.

**Гипотезы:**
- **H₀**: Сложные нейросетевые методы не превосходят традиционные методы (Tesseract OCR) по точности распознавания текста (WER, CER, IoU).
- **H₁**: Современные нейросетевые методы (CRNN, Transformer) обеспечивают лучшее качество распознавания текста по сравнению с классическими методами.

## **Функциональность**
- Распознавание текста с использованием Tesseract OCR.
- Применение методов предобработки изображений (бинаризация, фильтрация).
- Тестирование нейросетевых моделей для OCR.
- Оценка точности распознавания по метрикам WER (Word Error Rate), CER (Character Error Rate) и IoU (Intersection over Union).

## **Результаты**
- Графическое сравнение методов по метрикам WER и CER.
- Визуализация улучшений после применения предобработки изображений.
- Выводы о применимости нейросетевых методов в OCR.
